# 💳 Card Management System — Full Stack

A **full-stack fintech project** that manages **bank accounts** and **cards**, allowing users to create accounts, issue cards, track balances, and manage transactions.

---

## 📌 Features

* User registration and authentication
* Account management (create, update, delete)
* Card management (issue, activate/deactivate, view card info)
* Transaction history and balance tracking
* Admin panel to manage users and cards
* Responsive and user-friendly interface

---

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript (Bootstrap or Tailwind if used)
**Backend:** Spring Boot,PHP
**Database:** PostgreSQL
**Tools & Libraries:** Git & GitHub, Postman

---

## ⚡ Installation

1. **Clone the repository**

```bash
git clone https://github.com/username/card-management-system.git
```

2. **Navigate to the project folder**

```bash
cd card-management-system
```

3. **Install backend dependencies**

```bash
# For Spring Boot, ensure you have Maven or Gradle installed
./mvnw clean install   # or use gradlew if Gradle
```

4. **Set up the database**

* Make sure PostgreSQL is installed and running
* Create a database named `cms_db`
* **Note:** If any changes are made to the database, update your `application.properties` file accordingly

5. **Run the Spring Boot backend first**

```bash
./mvnw spring-boot:run   # or use gradlew bootRun
```

6. **Open the frontend**

[* Open http://localhost:8080/frontend/Login/signUp.php)

---


## 📖 Usage

* Register as a user or admin
* Create bank accounts
* Issue new cards
* Track balances and transactions
* Manage users and cards via admin panel
